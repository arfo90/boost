# Boostx

Boostx is code generator for ruby project based on required template

  - It helps to create basic files and folders (module, lib, etc)
  - Creating Rake test automatically 

# Command

```sh
$ gem install boostx
$ boostx init <project_name>
```


```sh
--project_name
  -lib
	-project_name
	-project_name.rb
  -Gemfile
```
